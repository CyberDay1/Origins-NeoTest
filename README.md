# NeoForge Multiversion (Stonecutter)

## Switch versions
```bash
./gradlew stonecutterSwitchTo1.21.1-neoforge
./gradlew build
```

Latest example:
```bash
./gradlew stonecutterSwitchTo1.21.10-neoforge
./gradlew build
```

## Notes
- Java 21 toolchain recommended.
- Vars available in templates: ${PACK_FORMAT}, ${LOADER_FILE}.
